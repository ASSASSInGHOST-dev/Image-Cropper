# Interactive Image Cropper

A simple image cropping tool built with Python, OpenCV, and Tkinter.

## Features

- Select images from your computer
- Draw a crop region using the mouse
- Real-time crop selection preview
- Instant cropped image display
- Supports JPG, JPEG, PNG, and BMP images

## Requirements

```bash
pip install opencv-python numpy
```

## Run

```bash
python main.py
```

## Controls

- Left Click + Drag → Select crop area
- Release Mouse Button → Crop image
- Press `X` → Exit

## Built With

- Python
- OpenCV
- NumPy
- Tkinter
